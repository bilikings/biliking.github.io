Action Group

这些是 IntelliJ IDEA 中内置操作的名称。您可以<add-to-group> 在插件的plugin.xml.

每个表显示一个组中的操作，其 ID 显示在 中 fixed-width font，以及其父组的 ID（供参考）及其显示名称。您可以在<add-to-group>的group-id属性中使用组 ID 。

表中的每一行代表一个动作，其 ID 可以在 <add-to-group>的relative-to-action 属性中使用。
